Чемпіонат має наступні номінації
Front-end developer
Верстка дизайн-макетів в HTML та CSS, програмування на JavaScript
Back-end developer
Програмування сервер-сайд систем сайтів. Основні мови: Python, Java, PHP, Ruby, ASP, JavaScript
Веб-дизайнер
Розробка дизайна веб-сайту, інтерфейсу веб-додатка тощо
Тестувальник
Тестування сайтів, веб-додатків, систем
Команда
Робота у команді над комплексним завданням
Навіщо брати участь у UA Web Challenge VII
  • оцінити себе
  • отримати пачку нових цікавих знайомств з колегами
  • знайти нові можливості (контакти, замовлення, тощо)
  • знайти нову роботу
  • гарний шанс перевірити себе
  • цікаві завдання з викликом - для тих, хто втомився від робочої рутини
  • отримати дозу позитиву під час проведення фіналу
  • порівняти та підняти свій рівень
Та, звичайно, як приємний бонус - призи переможцям
